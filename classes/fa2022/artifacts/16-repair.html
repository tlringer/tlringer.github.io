Getting Started

This class, we are going to implement proof repair as a plugin. Not the whole thing, just for a restricted class of changes. I've made a tutorial for this! Though it's fairly normal if the tutorial is a bit challenging. Please don't hesitate to ask for help.

Installation Instructions

You should have already cloned the tutorial plugin from Github. Now you will need to update it---run git pull. Then cd repair and build the second tutorial.

If you have any trouble, please post your questions in this class forum thread. Please also drop any installation tips you have in that thread.

If installation fails, and the class forums are no help, do not panic; we will try to match you with someone in class for whom installation worked.

You can find the exercise we will do together in class below.
